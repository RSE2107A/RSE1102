    Why It's Necessary¶

    ROS 2 uses CMake (for C++) and the ament build system under the hood. When you run:

    You are doing the following:

    • üß± Compiling your C++ source files

    Even if your project is written entirely in Python (which is not compiled), you still need to run colcon build. Here's why:

    • It ensures that the install/ directory is properly updated.
    • ROS 2 tools like ros2 run and ros2 launch look for packages in the install/ tree.

    Action Purpose

    This activates your latest build and registers it with your current terminal session.
